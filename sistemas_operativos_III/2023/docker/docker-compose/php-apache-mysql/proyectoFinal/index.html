El S.G.T - Sistema de Gestión de Trabajadores, fue creado para explicar el manejo de bases de datos en PHP usando PDO.

El software no utiliza patrones de diseño, solamente trabaja con los objetos de PDO para realizar el CRUD con la base de datos.

Este software se hizo solamente con fines educativos.
